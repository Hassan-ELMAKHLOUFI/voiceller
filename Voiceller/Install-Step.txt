1. Create and Import Database cloudpolly.sql
2. Upload and Unzip cloudpolly.zip to your hosting
3. Edit file .env and enter info your database
4. The default login name is: admin@example.com and default password is: admin12345
You can change it after loggin in in your profile settings.


*How to update*
Follow the Update in the documentation.


Before installation, your server must have following requirements to run the script properly

    PHP v7.4 or higher
    PHP Mbstring Extension
    PHP PDO Extension
    PHP JSON Extension
    PHP CURL Extension
    Support for MySQL v5.x
